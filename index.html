AgriSense Smart Agriculture System

ONOS SDN + Edge AI + Secure Cloud Connectivity + Autonomous Robotics for Precision Farming

Project Scope

AgriSense is a smart farming system designed for rural Sri Lanka using IoT sensors, Software-Defined Networking (ONOS SDN), secure IPsec tunneling, and edge AI to enable real-time crop monitoring, autonomous decision-making, and intelligent agricultural automation.

Research Problem

Rural agricultural environments suffer from unstable connectivity, lack of real-time monitoring, and inefficient bandwidth usage, leading to delayed irrigation control, poor crop health decisions, and limited automation capabilities for farming operations.

Proposed Solution

A Raspberry Pi-based SDN edge gateway powered by ONOS controller that manages IoT traffic, integrates secure StrongSwan IPsec tunnels over Azure cloud VMs, and supports AI-based crop monitoring using YOLO. The system also includes a 3D-printed robotic arm and an autonomous path-detecting robot for precision agriculture tasks.

Technologies Used

  • Python (Edge AI + SDN Controller)
  • Raspberry Pi 5
  • OpenFlow / ONOS SDN
  • Open vSwitch (OVS)
  • LoRa / ESP32 IoT Sensors
  • TensorFlow / YOLO (AI Detection)
  • StrongSwan IPsec VPN + Azure Virtual Machines
  • 3D Printed Robotic Arm
  • Autonomous Path-Detecting Robot

System Features

  • Real-time soil moisture, temperature, humidity monitoring
  • AI-based crop health and disease detection using YOLO
  • SDN-based traffic prioritization (QoS control via ONOS)
  • Secure IPsec tunneling between edge and Azure cloud
  • Autonomous robotic navigation for field inspection
  • 3D-printed robotic arm for precision farming tasks
  • Edge-based decision making for low-latency control
